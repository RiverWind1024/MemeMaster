## 3. 导航与路由

```dart
// go_router 定义
final router = GoRouter(
  initialLocation: '/',
  routes: [
    // 底部 Tab 导航
    StatefulShellRoute.indexedStack(
      branches: [
        // Tab 0: Gallery
        StatefulShellBranch(routes: [
          GoRoute(
            path: '/',
            builder: (_, __) => const GalleryScreen(),
            routes: [
              GoRoute(
                path: 'meme/:id',
                builder: (_, state) => MemeDetailScreen(
                  memeId: state.pathParameters['id']!,
                ),
              ),
            ],
          ),
        ]),
        // Tab 1: Search
        StatefulShellBranch(routes: [
          GoRoute(path: '/search', builder: (_, __) => const SearchScreen()),
        ]),
        // Tab 2: Folders
        StatefulShellBranch(routes: [
          GoRoute(path: '/folders', builder: (_, __) => const FolderScreen()),
        ]),
        // Tab 3: Settings
        StatefulShellBranch(routes: [
          GoRoute(
            path: '/settings',
            builder: (_, __) => const SettingsScreen(),
              routes: [
              GoRoute(path: 'models', builder: (_, __) => const ModelManagerScreen()),
              GoRoute(path: 'sync', builder: (_, __) => const SyncConfigScreen()),
              GoRoute(path: 'pipeline', builder: (_, __) => const PipelineConfigScreen()),
            ],
          ),
        ]),
      ],
    ),

    // 全屏路由（无底部导航栏）
    GoRoute(path: '/import', builder: (_, __) => const ImportScreen()),
  ],
);
```

## 4. 用户操作流程

### 4.1 首次使用流程

```
① 首次打开 → Gallery 空状态
    │
    ├──→ 点击 [📥 导入表情包]
    │       │
    │       ├── 从相册选择 → 选图 → 确认
    │       ├── 从 ZIP 导入 → 选文件 → 自动解压
    │       └── 从目录导入 → 选目录 → 递归扫描
    │
    ├──→ 导入完成 → 显示结果摘要
    │       │
    │       └── 缩略图显示在 Gallery (灰点=分析中)
    │
    └──→ 通知: "是否下载 AI 模型以获得语义搜索？"
            │
            ├── 是 → 下载模型 (进度条)
            └── 否 → 轻量模式 (仅颜色搜索)
```

### 4.2 搜索流程

```
② 搜索 "悲伤的猫"
    │
    ├── Tab 切换或点击 Gallery 顶部搜索栏
    ├── 输入 "悲伤的猫"
    ├── 可同时选择蓝色颜色芯片 (可选)
    │
    ├── 结果: 显示匹配度排序的 meme 网格
    │
    └── 点击某个结果 → Meme Detail 查看
```

### 4.3 颜色搜索流程

```
③ 点击 Search Tab
    │
    ├── 在颜色行选择一个蓝色系
    ├── 结果自动刷新为蓝色主色调 meme
    ├── 再加选白色 → 结果缩小为"蓝+白"配色
    │
    └── 再加文字 "悲伤" → 混合搜索
```

## 5. 状态处理

### 5.1 加载状态

```dart
// Riverpod AsyncValue 自动处理三种状态
final memeListProvider = FutureProvider.autoDispose((ref) async {
  return ref.read(memeRepositoryProvider).getAll();
});

// UI 层:
memeList.when(
  data: (memes) => MemeGridView(memes: memes),
  loading: () => const SkeletonGrid(),    // 骨架屏（灰色方块闪烁）
  error: (err, stack) => ErrorCard(
    message: '加载失败: $err',
    onRetry: () => ref.invalidate(memeListProvider),
  ),
);
```

### 5.2 分析进度状态

```dart
// AppBar 下方或 FAB 上方显示进度条:
// "正在分析 3 个表情包..."
// 点击可查看详情（正在分析哪个、已完成哪些）

// 单个缩略图的角标:
// - 灰点: pending（等待分析）
// - 蓝色旋转: processing（分析中）
// - 绿点: done（已完成）
// - 红点: failed（失败，点击可重试）
```

### 5.3 空状态

| 场景 | 显示 |
|------|------|
| 无任何 meme | 插画 + "导入你的第一个表情包" + 导入按钮 |
| 搜索无结果 | "没有找到相关表情包，试试其他关键词" + 颜色搜索建议 |
| 文件夹为空 | "这个文件夹还是空的，从其他文件夹移入或导入新 meme" |
| 同步未配置 | "配置 S3 同步以在多设备间同步你的 meme 收藏" |

### 5.4 错误状态

| 场景 | 处理 |
|------|------|
| 数据库错误 | 全屏错误页 + "重启 App" 按钮 |
| 存储空间不足 | 对话框提示 + 引导到存储清理页面 |
| 模型加载失败 | 显示错误原因 + "重试 / 重新下载" 选项 |
| 同步网络错误 | Toast "网络连接失败" + 自动重试 |
